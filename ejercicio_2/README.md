# Problema 2 — Detección de placas patente

## Descripción

Procesamiento de un conjunto de 12 imágenes (`img_1.jpg` … `img_12.jpg`) que muestran vehículos vistos desde adelante o desde atrás, cada uno con su correspondiente placa patente argentina modelo Mercosur.

Para cada imagen debemos:

- Detectar y segmentar la placa patente presente.
- Segmentar individualmente cada uno de los caracteres de la placa.
