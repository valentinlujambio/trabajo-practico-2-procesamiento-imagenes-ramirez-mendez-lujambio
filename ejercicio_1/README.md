# Problema 1 — Detección y clasificación de pastillas

## Descripción

Procesamiento automático de una imagen que muestra una **cinta transportadora industrial** con distintos tipos de pastillas, las cuales varían en forma, tamaño y color.

A partir de esa única imagen (`pills.png`) buscamos:

- Segmentar el área de la cinta donde se encuentran las pastillas.
- Detectar y segmentar cada pastilla de forma individual.
- Clasificar las pastillas por tipo y realizar un conteo total.
- Informar los resultados por consola y generar una imagen donde cada pastilla aparezca rotulada con su tipo e identificador.
